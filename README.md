Nepal House Price Prediction 🇳🇵🏠

A Machine Learning-powered web application that predicts house prices in Nepal using real-world property listing data scraped from Nepali real estate platforms.

This project combines:

Web Scraping
Data Cleaning
Exploratory Data Analysis (EDA)
Machine Learning
Streamlit Web Application

The dataset was created by scraping real property listings from Nepali real estate websites using Selenium.

🚀 Features

✅ Real Nepal property dataset
✅ Automated web scraping using Selenium
✅ Data preprocessing & cleaning
✅ Feature engineering using Regex
✅ Machine Learning price prediction
✅ Interactive Streamlit web app
✅ Nepal-focused real estate analysis

📊 Dataset

The dataset contains real Nepal house listings including:

Location
Bedrooms
Land Area (Aana)
Price
Property Description

Data was collected using Selenium-based web scraping.

⚙️ Machine Learning Workflow
1. Data Collection
Scraped real estate listings from Nepali property websites
2. Data Cleaning
Removed missing values
Extracted features using Regex
Converted prices into numeric format
3. Exploratory Data Analysis (EDA)
Price distribution
Location analysis
Area vs Price correlation
4. Model Training

Models experimented:

Linear Regression
Random Forest Regressor
5. Web Application
Built interactive prediction UI using Streamlit
🧠 Sample Prediction Features

The model predicts house price based on:

Location

Land Area
Bathrooms
Parking Availability
